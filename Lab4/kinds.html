Kinds of Testing



Remote Network

This test simulates the intruder launching an attack across the internet.The primary defences that must be defeated here are border firewalls, filtering routers and web servers.

Remote dial-up network

This test simulates the intruder launching an attack against the client's modem pools the primary defences that must be defeated here are user authentication schemes. These kinds of tests should be coordinated with the local telephone company

Stolen Laptop Computer

In this test the laptop computer of a key employee, such as a upper level manager or a trategist, is taken by the client without warning and given to the ethical hackers. They examine the computer for passwords stored in dial-up softwares, corporate information assets, personal information, and the like. Since many busy user will store their passwords on their machine, it is common for ethical hackers to be able to use this laptop computer to dial into the corporate internet with the owner's privilages.
